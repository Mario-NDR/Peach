## 目录结构说明
```
.
├── README.md               说明文档
├── dist                    打包文件
├── node_modules            依赖包
├── package.json            项目配置信息
├── postcss.config.js       postcss配置
├── src                     源代码
├── webpack-config          webpack插件和加载器
├── webpack.config.js       webpack配置文件
└── yarn.lock               yarn依赖管理
```

## 启动开发环境
### 安装依赖：
``` bash
$ yarn
```
或者
``` bash
$ npm install
```
### 启动开发环境：
``` bash
$ npm start
```
### 浏览器访问：[Demo](http://120.27.250.3:9955)

## 打包
``` bash
$ npm run build
```

## docker后端镜像
``` bash
$ docker build -t mario .
$ docker run -it -p 5000:5000 -d mario
```
